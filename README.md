# FakeStore E-commerce – Aplicación Web

Aplicación web colaborativa que consume la API pública de [Fake Store](https://fakestoreapi.com/) para mostrar un catálogo de productos, filtrarlos por categoría y gestionar un carrito de compras.

## Tecnologías utilizadas

- HTML5
- CSS3 / Bootstrap 5
- JavaScript (ES6+)
- Fetch API / APIs REST
- LocalStorage
- Git y GitHub

## Características

- Listado dinámico de productos traídos desde la API.
- Filtros por categoría (electronics, jewelery, men's clothing, women's clothing).
- Buscador por texto.
- Modal con detalle de producto.
- Botón **scroll to top**.
- Carrito de compras persistido en `localStorage` (desarrollado junto al equipo).

## Mi aporte en el proyecto

- Maquetación completa de la página (estructura HTML y estilos base).
- Integración con la API mediante `fetch` y modularización en `index.js` y componentes.
- Lógica de renderizado de tarjetas de productos y actualización del DOM según filtros.
- Implementación de navegación, búsqueda y UX básica.
